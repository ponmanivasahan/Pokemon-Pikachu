# Pokemon Pikachu

![Pokemon Pikachu image](./images/town.png)

Pokemon Pikachu is a small browser game built with HTML, CSS, JavaScript, Canvas, and GSAP. The project focuses on movement across a tiled map and a simple battle scene.

## Current Features

- Responsive canvas that scales to the browser window
- Top-down movement with `W`, `A`, `S`, and `D`
- Mobile movement buttons for touch devices
- Collision-based map navigation
- NPC dialogue areas
- Battle screen with HP bars and dialogue text
- Attack buttons for the current monster
- Basic battle animations and scene transitions

## Controls

| Key | Action |
| --- | --- |
| `W`, `A`, `S`, `D` | Move around the map |
| Mobile buttons | Move on touch devices |
| Mouse click | Use battle actions and advance dialogue |

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Canvas
- GSAP
